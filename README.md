# Super-Store-Sales-Analysis-15-Day-Forecast-Power-BI-
Project Overview
This project analyzes Super Store sales data and presents insights through interactive dashboards, along with a 15-day sales forecasting model.
The aim is to support business decision-making by combining historical analysis with predictive insights.

•	Total Sales: 1.6M
